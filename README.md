# termux
termux
安装步骤：<br>
pkg install -y curl <br>
curl -o zfx https://raw.githubusercontent.com/ZFXLYN/termux/master/zfx.sh <br>
mv -f zfx $PREFIX/bin/ <br>
chmod +x $PREFIX/bin/zfx <br>
启动命令：zfx
